# video_dl
download video from online video sites.

# Dependence
## Python Version
```
3.8+
```
## Third Party Libraries
```bash
pip3 install -r requirements
```
## System Dependence
- [ffmpeg](https://ffmpeg.org/)

# Usage
## Normal Usage
```bash
python3 bilibili.py https://www.bilibili.com/video/BV1qP4y1x7v3
```
## Help
```bash
python3 bilibili.py -h
```
